Project Overview

How I integrated four separate assignments into one cohesive web page

What This Page Contains

This integrated page combines my previous four data visualization assignments (A1-A4) into a single, unified web interface. Each assignment demonstrates different aspects of interactive data visualization using D3.js and modern web technologies.

  • A1: API-driven interactive data table
  • A2: SVG transformations and manipulations
  • A3: Interactive chart with transitions
  • A4: Complex grouped data visualizations

Integration Approach

To merge the four assignments, I created a unified HTML structure with consistent styling and navigation. Each assignment is loaded in its own container using iframes to prevent CSS and JavaScript conflicts. I implemented a tabbed navigation system and responsive design to ensure a smooth user experience across all devices.

1 Create consistent layout & navigation
2 Embed each assignment in isolated containers
3 Implement responsive design for all screen sizes
4 Add overview and difficulty explanation

Challenges & Solutions

Main challenges faced during integration:

  • CSS Conflicts

    Each assignment had its own CSS rules that conflicted when combined. Solution: Used iframes to isolate each assignment's styling.

  • Responsive Design

    Different assignments had different layout requirements. Solution: Created a flexible container system with adjustable iframe heights.

  • Interactive Elements

    Assignment 3 had complex interactions that needed to work within the iframe. Solution: Ensured proper event propagation and iframe sizing.

Hardest Assignment to Integrate

Assignment 3 was the most challenging to integrate due to its complex interactive elements, transitions, and tightly coupled code structure. The visualization relied heavily on specific DOM element positions and event listeners that needed careful handling within the iframe environment.

Key takeaways from integrating four separate assignments
